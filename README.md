RedFlag is a SQL-based fraud detection project built using a dataset of nearly 200,000 digital payment transactions from a simulated Indian payment aggregator. The objective was to identify suspicious user and merchant behavior by writing SQL queries for different fraud patterns without using Python or machine learning. This project helped me improve my SQL skills by working with real-world concepts such as aggregations, subqueries, Common Table Expressions (CTEs), and window functions while solving practical fraud detection problems
